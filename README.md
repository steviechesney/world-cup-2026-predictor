\# World Cup 2026 Predictor



Excel-based World Cup 2026 prediction and leaderboard system designed to manage prediction competitions between friends and colleagues.



The workbook automates scoring, leaderboard rankings and match result updates throughout the tournament group stage.



\## Features



\- Match prediction templates for multiple players (up to 85)

\- Automated scoring system

\- Dynamic leaderboard updates

\- Correct score and correct result logic

\- Centralised fixtures and results management

\- Excel formula-driven automation



\## Scoring System



\- Correct score and result = 3 points

\- Correct result only = 1 point



\## Workflow

Prediction templates distributed

↓

Players submit predictions

↓

Predictions copied into results sheet

↓

Match scores updated

↓

Points calculated automatically

↓

Leaderboard updates dynamically
